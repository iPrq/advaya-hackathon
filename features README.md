 Camera-less Fall Detection: We engineered a non-intrusive safety system using Wi-Fi CSI (Channel State Information). By running real-time WebSocket subcarrier analysis through a Python inference engine, Aegis detects physical falls without a single camera, instantly alerting caretakers.

 Vision-Powered Safety Guardrails: We integrated Gemini 2.5 Flash Vision to ingest high-density medical documents, automatically structuring medication schedules and proactively flagging critical contraindications.
 
 Agentic Web Intelligence: We built a specialized browser extension paired with a context-aware RAG chatbot, allowing users to "interrogate" dense medical literature and extract simplified, personalized summaries. 

 Zero-Barrier Accessibility: Powered by Groq TTS for ultra-low latency voice synthesis and a cross-lingual translation layer (English/Kannada) to ensure critical medical information is accessible to native speakers. 

 Community Connectivity: An interactive mapping interface that helps patients discover and connect with local support groups and localized medical resources.

The Architecture: Built as a hybrid full-stack solution optimized for native iOS and Android deployment using Next.js 16, FastAPI, Capacitor, WebSockets, and TypeScript.
